1、使用JavaScript将图片拷贝进画布
将图片放入画布里，使用canvas元素的drawImage方法即可： 
  1. // Image to canvas; returns new canvas element
  2. function ImageToCanvas(image) {
  3. 	var canvas = document.createElement("canvas");
  4. 	canvas.width = image.width;
  5. 	canvas.height = image.height;
  6. 	canvas.getContext("2d").drawImage(image, 0, 0);//0, 0参数画布上的坐标点，图片将会拷贝到这个地方
  7. 	return canvas;
  8. }
2、用JavaScript将画布保存成图片格式
画布上的作品已经处理完成，可以用toDataURL将canvas数据转换成图片格式（png）：
 
  1. // Canvas to an image
  2. function canvasToImage(canvas) {
  3. 	var image = new Image();
  4. 	image.src = canvas.toDataURL("image/png");
  5. 	return image;
  6. }
