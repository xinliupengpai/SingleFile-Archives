[Fixed] INET_E_RESOURCE_NOT_FOUND Error

Lillian Lai

Last Updated: 3 years ago
0

If you see the Error Code: INET_E_RESOURCE_NOT_FOUND on your computer, don’t worry! You can fix the error.

This is a common error in Windows, especially in your browser after Windows Update. This article introduces some solutions to fix the INET_E_RESOURCE_NOT_FOUND error in Windows.

Try these fixes:

Here are the solutions you can try. You don’t need to try them all; just work you way down until it solves your INET_E_RESOURCE_NOT_FOUND error.

  1. Disable TCP Fast Open feature on Microsoft Edge
  2. Update your network driver
  3. Reinstall Microsoft Edge
  4. Flush DNS
  5. Use a VPN

Fix 1: Disable TCP Fast Open feature on Microsoft Edge

TCP Fast Open is a feature introduced by Microsoft to improve the performance and security, but disabling it won’t impact the proper work of your computer and browser. To do so, check these steps:

1) Launch Microsoft Edge, type about:flags in the URL address field.

2) In the Networking section, uncheck the box next to Enable TCP Fast Open.

3) Restart your computer, and open your Microsoft Edge.

This should fix the INET_E_ RESOURCE_NOT_ FOUND error.

Fix 2: Update your network driver

The network driver issue in your computer can cause the INET_E_RESOURCE_NOT_FOUND error, so you should check your network driver and update it to the latest version.

There are two ways to update drivers: manually and automatically.

Manually update driver – You can manually update the network driver by searching the correct driver that is matched with your Windows OS on the manufacturer’s website, and install it in your computer. This requires the time and computer skills.

Automatically update driver – If you don’t have time or patience to manually update drivers, you can do that automatically with Driver Easy.

Driver Easy will automatically recognizes your system after scanning, then find and install automatically the latest drivers for you, which saves your time and enhances your Windows performance.

1) Download and install Driver Easy.

2) Run Driver Easy and click Scan Now. Driver Easy will then scan your computer and detect any problem drivers.

3) Click the Update button next to a flagged network driver to automatically download and install the latest audio driver (you can do this with the FREE version).

Or click Update All to automatically download and install the all the latest correct drivers that is missing or outdated in your system (This requires the Pro version. You will be prompted to upgrade when you click Update All).

4) Restart your computer and check whether your problem is solved.

Fix 3: Reinstall Microsoft Edge

This method works for many users, but before uninstall Microsoft Edge, you should backup your files.

Step 1: Backup your favorites

1) On your keyboard, press the Windows logo keyand R at the same time to invoke the Run box.

2) Copy and paste %LocalAppData%PackagesMicrosoft.MicrosoftEdge_8wekyb3d8bbweACMicrosoftEdgeUserDefault and click OK.

3) Right click on the DataStore folder and select Copy.

3) Go to your desktop, right click on any empty space, and select Paste.

Now you’ve back up your favorites.

Step 2: Reinstall Microsoft Edge

1) Close Microsoft Edge, and open File Explorer.

2) Click the View tab and check the box next to Hidden items.

3) Go to C:UsersUserNameAppDataLocalPackages. Kindly note that you should replace UserName with your user name on your computer.

4) Right click on the Microsoft.MicrosoftEdge_8wekyb3d8bbwe folder, and delete it.

5) Click Yes to confirm. It may take a while to proceed. If a dialog prompts, click Yes to confirm. If the system refused to delete some of the files, click Skip.

6) After completed, restart your computer.

7) Type PowerShell in the search box on your desktop, then right click Windows PowerShell and select Run as Administrator.

8) In Windows PowerShell, type cd C:, and press Enter. Then type cd C:usersUserName in the PowerShell (Replace UserName with your user name), and press Enter.

9) Copy and paster the following command, and press Enter.

 Get-AppXPackage -AllUsers -Name Microsoft.MicrosoftEdge | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)AppXManifest.xml” -Verbose}

10) Once completed, close Windows PowerShell, and restart your computer.

Fix 4: Flush DNS

You can try flush the DNS and try to connect to your network again to fix the INET_E_RESOURCE_NOT_FOUND error.

1) Type cmd in the search box on your desktop, right click Command Prompt (or cmd), and select Run as administrator.

2) Copy and paste the following command in Command Prompt, and press Enter.

ipconfig /flushdns

3) You should see a message Windows IP configuration successfully flushes the DNS Resolver Cache.

4) Type exit and press Enter.

5) Restart your PC and launch your Microsoft Edge.

Fix 5: Use a VPN

If you’re still having this error after trying all the methods above, you can use a VPN. Since this is a DNS error, your DNS settings for your network are not configured correctly. If that’s the case, using a VPN will connect to a new DNS server and then it resolves your DNS problem, so your error is fixed.
