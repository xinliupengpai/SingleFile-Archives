ERROR: The partition with /var/lib/mysql is too full! failed!

NND 隔段时间就出现一次  不知道该怎么个彻底解决

暂时解决办法：

root@li611-147:~# cd /var
root@li611-147:/var# rm -rf log

删除日志文件,然后重启mysql

root@li611-147:~# /etc/init.d/mysql start
