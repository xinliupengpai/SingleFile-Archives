Java入门教程：最全面最简单的Java基础教程

Java是一种跨平台的语言，一次编写，到处运行，在世界编程语言排行榜中稳居第二名（第一名是C语言）。

Java用途广泛，可以用来开发传统的客户端软件和网站后台，也可以开发如火如荼 Android 应用和云计算平台。

本教程针对有编程基础（至少了解C语言）的读者，阅读完所有文章会让你达到Java中级水平。零基础的读者可以先查看C语言教程，了解编程的基本概念。
